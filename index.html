passtime.js

  • Docs

Passtime.js in theory enables simple, fast and secure password generation based on the user's unique typing style.

The difficulty in creating random passwords lies in the fact that computers themselves cannot generate truly random numbers - this is where the user comes in. By using the timings of a user's keystrokes, passtime.js generates a string based on the random noise of their input.

This concept is based on the idea of using mouse movement [cf. https://entima.net/random] and other user randomness to achieve secure passwords, although this method is designed to be more user-experience friendly and implementable in web forms.

Example

Usage

· via HTML

To use passtime.js via HTML, simply add the data-passtime-target-id attribute to an input element. The value of this attribute should be the id of the password element that passtime.js should output to.

For instance,

<input id='username' type='text' data-passtime-target-id='password'>
<input id='password' type='text'>

will recreate the example above

· via Javascript

You can also use passtime.js via javascript by calling

new passtime.Control(inputElement, outputElement)

where inputElement is the HTMLElement to be used as input, and outputElement is where passtime.js will output the generated password.
